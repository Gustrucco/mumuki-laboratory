Mumuki Atheneum
================

> Exercises solving webapp for the Mumuki platform

## About

Atheneum is a multitenant Rails webapp for solving exercises, organized in terms of chapters and guides.

## Running

```bash
MUMUKI_AUTH0_CLIENT_ID=...\
MUMUKI_AUTH0_CLIENT_SECRET=... \
MUMUKI_AUTH0_DOMAIN=... \
rails s
```
